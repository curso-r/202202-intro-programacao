TCC Curso R - Introdução a programação com R
Aluna: Laila Viana

O trabalho final escolhido consiste em criar um acesso ao meu spotify e analisar os dados dos 
top50 artistas e top50 músicas mais ouvidas desde a criação do meu spotify.

Para isso utilizei três pacotes, o spotifyr que realiza o acesso ao meu spotify, o tidyverse
que possibilita a manipulação dos dados e o lubridate que me permite trabalhar com datas.

Dentro dessa pasta, serão disponibizados os seguintes arquivos:
- projeto_final.Rproj (arquivo Rproj)
- TCC-1.R (arquivo .R sobre como foi realizada a obtenção dos dados)
- TCC-2.R (arquivo .R sobre como foi realizada a manipulação dos dados)
- top_artists_laila.RData (arquivo contendo os top artistas gerados com as minhas credenciais do spotify que será manipulado no arquivo TCC-2)
- top_tracks_laila.RData  (arquivo contendo as top músicas geradas com as minhas credenciais do spotify que será manipulado no arquivo TCC-2)
- readme.txt (esse arquivo contendo uma breve explicação sobre o projeto e seus componentes)
- spotify_credentials.txt (minhas credenciais do spotify, resolvi deixar num arquivo a parte para simples conferencia caso seja necessario, uma vez que
os dados que irei analisar já estão salvos nessa pasta. Sendo assim, pensei que seria melhor não compartilhar dentro do arquivo .R)

Demais informações deixarei juntamente com o código nos arquivos .R para que fique claro qual a finalidade de cada função utilizada.
